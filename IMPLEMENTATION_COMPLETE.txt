==================================================
RUNNING THE APPLICATION
==================================================

1. Database Setup
   - Execute /database/schema.sql in Supabase

2. Start Backend
   cd backend && npm run dev
   http://localhost:8080

3. Start Admin Portal
   cd admin-portal && npm run dev
   http://localhost:3000

4. Start Vendor Portal
   cd vendor-portal && npm run dev
   http://localhost:3001

5. Start Customer Portal
   cd customer-portal && npm run dev
   http://localhost:3002
