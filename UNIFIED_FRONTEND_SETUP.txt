AUTHENTICATION FLOW
==========================================

1. User goes to http://localhost:3000/login
2. Selects role (Admin, Vendor, or Customer)
3. Enters email and password
4. Clicks "Sign In"
5. Backend validates credentials against Supabase
6. Token stored in localStorage
7. User redirected to appropriate portal:
   - Admin â†’ /admin
   - Vendor â†’ /vendor
   - Customer â†’ /
8. Navigation shows user name
9. User can click "Logout" to return to login
